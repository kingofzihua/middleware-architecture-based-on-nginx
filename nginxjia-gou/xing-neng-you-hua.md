#### ab接口压力测试工具
##### 安装
```shell
    yum install httpd-tools
```
##### 使用
```shell
    ab -n 2000 -c 2 http://127.0.0.1/
    # -n 总的请求数
    # -c 并发数
    # -k 是否开启长连接
```
